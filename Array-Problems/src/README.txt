25 Java Problem Question in Arrays.
===============================================================================================
1. Write a program to find the maximum element in an array.
2. Implement a program to find the minimum element in an array.
3. Create a program to calculate the sum of all elements in an array.
4. Write a program to calculate the average of elements in an array.
5. Implement a program to reverse an array.
6. Create a program to find the second largest element in an array.
7. Write a program to find the second smallest element in an array.
8. Implement a program to sort an array in ascending order.
9. Create a program to sort an array in descending order.
10. Write a program to check if an array is sorted in ascending order.
11. Implement a program to check if an array contains a specific element.
12. Create a program to remove duplicate elements from an array.
13. Write a program to find the frequency of each element in an array.
14. Implement a program to find the sum of elements at even indices in an array.
15. Create a program to find the product of elements at odd indices in an array.
16. Write a program to find the maximum difference between two elements in an array.
17. Implement a program to rotate an array to the right by a given number of positions.
18. Create a program to find the pair of elements with the maximum sum in an array.
19. Write a program to find the missing number in a given range in an array.
20. Implement a program to find the intersection of two arrays.
21. Create a program to find the union of two arrays.
22. Write a program to merge two sorted arrays into a single sorted array.
23. Implement a program to remove the duplicates and return a new array.
24. Create a program to find the maximum subarray sum in an array.
25. Write a program to rearrange the positive and negative numbers in an array.
